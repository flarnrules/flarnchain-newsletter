Flarnchain Newsletter Facelift!

That's right. The newsletter got a facelift. I wrote a whole bunch of Python scripts and wired them all together so that I can send emails though g-mail from the Command Line Interface on any computer. Since all of the media files need to be stored in a content delivery service to send the emails, I now can write the newsletter from anywhere, on any device, so long as I have access to the internet.

Here's a screenshot of my project directory tree. I spent way too much time agonizing about how to organize the files... That I never accomplished anything. This oranization is most definitely not the best, but I'm trying to learn to not let perfect be the enemy of good enough

I also get to benefit from the automation benefits that come from having a back-end database that stores email addresses, rather than just keeping the subscriber list in an excel spreadsheet. This may not seem like a lot, or kind of pointless, as I could have just used a service like Survey Monkey to do all of this for me, but I really wanted to make sure that I understand as much of the tech stack as possible that is used by the Flarnchain ecosystem. The more knowledge I have about the interworkings of content delivery on the internet, the better I will be prepared to grasp those concepts as they relate building the Flarnchain blockchain itself.

Here's a screenshot of the main chunk of code that actually sends the newsletter out. There's some improvements that I can make to this which would make the process of sending more streamlined, but for now, I'm pleased with what I've built.

So not only have I learned a lot through the process of building this, but I'm also gradually building out the infrastructure that will be used for bigger and better things to come. I'm really excited and happy with this accomplishment.
